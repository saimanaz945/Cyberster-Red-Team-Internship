# Week 02 – Cyberster Red Team Internship

## Overview

During Week 02, I performed advanced network reconnaissance and vulnerability assessment using Nmap on the authorized Metasploitable 2 target machine.

## Tasks Completed

### Task 01 – Advanced Port Scanning
- TCP Connect Scan (-sT)
- SYN Stealth Scan (-sS)
- UDP Scan (-sU)
- Timing Templates (-T2 & -T4)

### Task 02 – Service Fingerprinting & Banner Grabbing
- Version Detection (-sV)
- Aggressive Scan (-A)
- Manual Banner Grabbing (Netcat)
- SMB Enumeration

### Task 03 – Nmap Scripting Engine (NSE)
- HTTP Enumeration
- Discovery Scripts
- Safe Scripts
- Vulnerability Detection
- CVE Mapping

### Task 04 – Stealth & Firewall Evasion
- Packet Fragmentation (-f)
- MTU Manipulation (--mtu)
- Decoy Scans (-D)
- Source Port Spoofing (--source-port)

## Tools Used

- Nmap
- Netcat
- Enum4linux
- Kali Linux
- Metasploitable 2

## Learning Outcomes

- Performed advanced port scanning techniques.
- Identified running services and software versions.
- Detected known vulnerabilities using NSE scripts.
- Mapped findings to CVE identifiers.
- Practiced stealth scanning and firewall evasion techniques.
- Improved practical penetration testing skills.
